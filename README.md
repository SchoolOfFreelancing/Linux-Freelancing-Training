## 🎓 Program Overview

**Linux Freelancing Training** is a 100% hands-on, practical program that takes participants through real Linux system administration work, structured specifically to prepare them for paid freelance jobs on Guru, Freelancer, and Upwork. This is not a certificate course — it is a **guaranteed-income-track** program tied to marketplace bidding and direct client acquisition.

## ✅ Guaranteed Income Conditions

Guaranteed income is based on two conditions:

1. **Full attendance.** Attend all training regularly. Do not leave training due to personal issues or technical glitches.
2. **Active marketplace bidding.** Purchase the necessary bids/connects on the freelance marketplace to apply for jobs.

## 💻 System & Participation Requirements

**Hardware:**
- Ubuntu Desktop (24.04 / 26.04 LTS)
- 64-bit dual-core processor (4 GHz)
- Minimum 8 GB RAM (16 GB recommended)
- At least 200 GB available disk space

**Participation:**
- Reliable fiber-optic internet connection
- Real IP address or server access
- Registered domain
- Valid government-issued ID
- Portable device for instant client communication
